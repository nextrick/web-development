INTRODUCTION

HTML is designed to create hypertext documents. Hypertext refers to text that contains links to other documents or resources, allowing users to navigate between them by clicking on the links.
These links can connect different web pages, sections within the same page, or multimedia content like images, videos, and audio files.
Furthermore, HTMl uses a markup language to annotate text within a document to indicate the structure and semantics of the content. This markup is achieved through the use of tags, which are enclosed in angle brackets(< >). Tags are used to define elements such as headings, paragraphs, lists, images
